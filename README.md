# Kiongozi Connect

**Jua Kiongozi Wako Amefanya Nini Leo**

Connecting Leaders and Citizens

## Brand Promise
Transparency. Progress. Connection.

## Vision
A national citizen-leader engagement platform starting in Meru and expanding across Kenya.

## Mission
A digital platform connecting citizens and leaders where citizens track what leaders have done today, this week and this month while leaders showcase projects, meetings and achievements.

## Technology Stack
- **Backend**: Supabase (Database, Auth, Storage, Realtime)
- **Frontend**: HTML/CSS/JavaScript
- **Deployment**: Vercel

## Features
- Citizen and Leader accounts
- Daily/Weekly/Monthly feeds
- Project management
- Meeting announcements
- Community groups
- Live sessions
- Social features (likes, comments, follows)
- Notifications
- Analytics dashboard
- Subscription plans

## Quick Start

```bash
git clone https://github.com/mutegiphilip28/Kiongozi-Connect.git
cd Kiongozi-Connect
npm install
npm run dev
```
